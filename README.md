# 📝 React Todo App (Tailwind CSS)

A simple and modern Todo List application built using **React + Vite + Tailwind CSS**. This project helps in understanding core React concepts like components, state management, props, and event handling.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* 📋 Display task list
* 🎨 Clean UI with Tailwind CSS
* ⚡ Fast development using Vite

---

## 🛠️ Tech Stack

* **Frontend:** React (with Hooks)
* **Build Tool:** Vite
* **Styling:** Tailwind CSS

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone <https://github.com/Utkarshende/react-todo-app.git>
cd react-todo
```

### 2. Install dependencies

```
npm install
```

### 3. Run the development server

```
npm run dev
```

App will run on:

```
http://localhost:5173/
```

---

## 🎨 Tailwind CSS Setup

Make sure Tailwind is configured:

### Install Tailwind

```
npm install -D tailwindcss@3 postcss autoprefixer
npx tailwindcss@3 init -p
```

### Add to `index.css`

```
@tailwind base;
@tailwind components;
@tailwind utilities;
```
